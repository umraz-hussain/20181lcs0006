HTML stands for Hypertext Markup Language
It is used to create web pages
It describes the structure of the web page
HTML is not a programming language.

CSS stands for Cascading Style Sheets
CSS describes how HTML elements are
to be displayed on screen, paper, or in other media
CSS saves a lot of work

JavaScript is a scripting language used to create and control dynamic website content.
It is a high-level, often just-in-time compiled,and multi-paradigm.JavaScript is most commonly used as a client side scripting language
